# PEP Assignments

This repository contains Java assignments for PEP (Programming Enhancement Program).

## Folder Structure

```
├── Assignment1/
│   ├── ATM.java           - ATM menu system using switch-case
│   └── VowelChecker.java  - Vowel/Consonant checker with loop
│
├── Assignment2/
│   ├── Student.java              - Abstract class with encapsulation
│   ├── EngineeringStudent.java   - Extends Student
│   ├── MedicalStudent.java       - Extends Student  
│   ├── SportsParticipant.java    - Interface
│   ├── CulturalParticipant.java  - Interface
│   ├── AllRounderStudent.java    - Multiple inheritance via interfaces
│   └── Main.java                 - Demonstrates polymorphism & late binding
│
└── Assignment3/
    ├── MyList.java        - Generic list interface
    ├── MyArrayList.java   - Array-based implementation
    ├── MyLinkedList.java  - Linked-list implementation
    └── Main.java          - Wildcards & exception handling demo
```

## Assignment Details

### Assignment 1: Switch-Case & Loops
- ATM Menu System
- Vowel/Consonant Checker

### Assignment 2: OOP Concepts
- Encapsulation, Abstraction, Inheritance
- Interfaces, Multiple Inheritance
- Polymorphism, Late Binding
- static and final keywords

### Assignment 3: Custom List Framework
- Generic Interface Design
- Array-based List Implementation
- Linked-list Implementation
- Exception Handling

## How to Run

Open any `.java` file in VS Code/IntelliJ and click the Run button.
